My Project

Leaf Website

Badan Pendidikan dan Pelatihan Pemeriksaan Keuangan Negara BPK RI

Leaf merupakan website yang berisi mengenai kegiatan yang meliputi diklat BPK, workshop dan kegiatan lain yang dibuat untuk pegawai internal Badiklat. Selain itu terdapat juga kegiatan lain seperti workshop dan pelatihan yang melibatkan pihak eksternal. Website ini dibuat menggunakan Laravel. Tech stack : Laravel

BidTech Website

Badan Pendidikan dan Pelatihan Pemeriksaan Keuangan Negara BPK RI

BidTech merupakan website Badiklat yang digunakan untuk penyimpanan data per masing-masing divisi yang ada di BPK. Website ini menggunakan akses login SSO yang akan digunakan untuk pegawai internal. Website akan dikelola kepada admin masing-masing divisi. Dan berikut adalah tampilan user interface website yang dirancang menggunakan Figma. PLatform : Figma

Loveliness App

Universitas Brawijaya - 2nd semester project assignment

Loveliness merupakan aplikasi kecantikan yang dapat diakses oleh semua user di Indonesia. Aplikasi ini memiliki dua kategori yaitu skincare dan makeup. Didalam aplikasi terdapat sekali berbagai macam produk yang tersedia dan pengguna dapat membeli hanya melalui aplikasi. Dan berikut adalah tampilan user interface website yang dirancang menggunakan Figma. Platform : Figma

Noempak Travel Website

Universitas Brawijaya - 3nd semester project assignment

Noempak Travel merupakan website pemesanan travel yang dapat diakses oleh banyak user. Website ini berisi beberapa wisata dan paket travel yang dapat dipesan oleh user. Website ini dibuat menggunakan figma sebagai user interface, PHP sebagai bahasa pemrograman dan menggunakan sql untuk database user. Tech stack : PHP & Xampp

Sistem Informasi RS Pelita Harapan

Universitas Brawijaya - 2nd semester project assignment

Sistem Informasi RS Pelita Harapan merupakan website yang digunakan untuk keperluan internal rumah sakit. Website ini dibuat dengan Java yang berisi mengenai database rumah sakit mulai dari data pasien sampai dengan data internal pegawai rumah sakit. Tech stack : Java

Travel Mobile App

Universitas Brawijaya - 3nd semester project assignment

Travel Mobile App merupakan aplikasi berbasis mobile yang digunakan untuk melakukan pemesanan tiket destinasi wisata. Aplikasi ini dibuat menggunakan flutter dan firebase sebagai database user. Tech stack : Flutter & Firebase
